Compile unittests with makefile:<br/>
These test the functions that their titles describe: getCost(), supplyCount(), fullDeckCount(), and isGameOver()<br/><br/>

Compile:<br/>
make getcost<br/>
make supplycount<br/>
make fulldeck<br/>
make isgameover<br/>

Run:<br/>
./getcost <br/>
./supplycount<br/>
./fulldeck<br/>
./isgameover<br/>
